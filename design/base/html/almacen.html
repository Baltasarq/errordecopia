Almacén


La pequeña estancia está rodeada de estanterías, en la que se agolpan varios objetos, de todos los tipos. Puedes ver una palanca, una llave inglesa, un destornillador, una linterna, y un martillo. Un pequeño armario se sitúa sobre una de las estanterías.

    Objetos:
  • armario[stc](Se trata de un pequeño panel que se puede deslizar mediante un pequeño pomo.)
  • batería(Una batería estándar de Litio-Iones.)
  • destornillador(Un destornillador de punta plana y mango amarillo.)
  • estantería[stc](No tiene demasiado de especial. Son de color blanco, el mismo que las paredes. Probablemente sean del mismo material.)
  • linterna(De plástico negro, con pantalla de cristal. Tiene un pequeño compartimento para una batería.)
  • llave inglesa(Pues sí, una llave inglesa.)
  • martillo(Es un martillo sin uñas.)
  • palanca(Se trata de una palanca de acero inoxidable.)

    Salidas:
  • ESTE hacia Encrucijada
